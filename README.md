## 프로젝트 설명

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## 설치

```bash
$ npm install
```

##실행방법

```bash
# 서버 실행
$ npm run start

# 개발 모드
$ npm run start:dev

# 프로덕션 모드
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
